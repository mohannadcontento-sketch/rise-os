## 📱 تطبيق الموبايل (PWA — يعمل بدون إنترنت)

التطبيق **جاهز كـ PWA** ويمكن تثبيته مباشرة:

### طريقة التثبيت على الموبايل:
1. افتح الموقع في **Chrome** (أندرويد) أو **Safari** (آيفون)
2. اضغط على **"إضافة إلى الشاشة الرئيسية"** من القائمة
3. التطبيق سيشتغل **بدون إنترنت** بالكامل
4. كل البيانات محفوظة محلياً على الجهاز

## 💻 تطبيق كمبيوتر EXE (Tauri)

### المتطلبات:
- **Rust**: https://rustup.rs
- **Node.js**: مثبت مسبقاً
- **Windows**: Visual Studio Build Tools

### خطوات البناء:

#### 1. تثبيت Tauri CLI:
```bash
cargo install tauri-cli
```

#### 2. الانتقال لمجلد المشروع:
```bash
cd riseos
```

#### 3. بناء EXE:
```bash
# للتصحيح
cargo tauri dev

# للإنتاج (ينتج ملف .exe في target/release/bundle/msi/)
cargo tauri build
```

#### 4. النتيجة:
- `target/release/bundle/msi/RiseOS_1.0.0_x64_en-US.msi` — ملف التثبيت
- `target/release/bundle/nsis/RiseOS_1.0.0_x64-setup.exe` — EXE مباشر
- `target/release/RiseOS.exe` — الملف التنفيذي

### ملاحظات:
- أول بناء يستغرق 5-10 دقائق (يحمّل مكتبات Rust)
- البنيات التالية أسرع بكثير
- الحجم النهائي ~10-15 ميجابايت (أخف بكثير من Electron)

## 📋 ملاحظات تقنية

### بنية أوفلاين:
- **Service Worker** (`public/sw.js`): يخزّن الملفات الثابتة وردود API
- **IndexedDB** (`src/lib/offline-db.ts`): قاعدة بيانات محلية لكل الوحدات
- **Sync Manager** (`src/lib/sync-manager.ts`): يزامن البيانات عند عودة الاتصال

### استراتيجيات التخزين المؤقت:
- الملفات الثابتة (JS, CSS, صور): **Cache-First** (سريع أوفلاين)
- API endpoints: **Network-First** (بيانات حديثة مع fallback)
- الصفحة الرئيسية: **Stale-While-Revalidate**

### البلوتوث:
- بروتوكول نصي مخصص: `RISEOS:length:json`
- fallback تلقائي لـ Web Share API
- البيانات المستلمة تُخزّن في IndexedDB
